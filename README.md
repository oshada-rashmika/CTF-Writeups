# 🏴‍☠️ The GhostNet Archives: CTF Writeups

Welcome to my personal stash of Capture The Flag (CTF) writeups! 

By day, I'm a regular CS undergrad surviving on caffeine and assignments. By night? I'm cruising through vulnerable networks like a 90s JDM racer on a midnight mountain pass, treating every target box like a hit contract waiting to be closed. 🏎️💨😈

Consider this repository the documentation of my villain arc. Expect detailed walkthroughs, exploit scripts, and a whole lot of hijacked cookies as I operate from the shadows and break things to understand how they work. 🥷✨

## 🗂️ What's in the Garage?

Here you'll find writeups broken down by category. Whether you're a beginner trying to learn the ropes or a seasoned hacker looking for a specific exploit, there's something here for you.

* 🌐 **Web Exploitation:** Session hijacking, SQL injections, XSS, and making web apps cry.
* 🔐 **Cryptography:** Breaking bad math and decrypting the "un-decryptable."
* 🛠️ **Reverse Engineering:** Tearing apart binaries to see what makes them tick.
* 💻 **Pwn / Binary Exploitation:** Buffer overflows and memory corruption madness.
* 🕵️ **OSINT & Forensics:** Tracking digital footprints and extracting hidden secrets.

## 📖 How to Read These Writeups

I write these guides to be as beginner-friendly and fun as possible. You won't just find the final payload here; I walk through:
1.  **The Recon:** How I approached the target.
2.  **The Vulnerability:** The underlying flaw in the code or logic.
3.  **The Exploit:** The exact steps (and scripts) used to secure the flag.
4.  **The Fix:** How developers can actually patch the hole so it doesn't happen again.
